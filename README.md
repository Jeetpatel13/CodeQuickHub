## CodeQuickHub

CodeQuick is a handy cheat sheet website for web developers, providing quick references for HTML, CSS, and JavaScript.

##Features

•	HTML Cheat Sheet – Get a list of common HTML tags and attributes.
•	CSS Cheat Sheet – Quick reference for selectors, properties, and layout techniques.
•	JavaScript Cheat Sheet – Common JS functions, events, and syntax examples.

##Technologies Used

•	HTML – For structuring the content.
•	CSS – For styling the website and making it visually appealing.
•	JavaScript – For interactivity, such as form handling or dynamic content.
